Class SqlServerDbContextFactory<TDbContext, TMigrationAssembly>

Represents a factory for creating instances of TDbContext configured to use SQL Server as the database provider. This abstract class provides methods to configure the DbContext using a connection string and optional additional SQL Server-specific configuration.
